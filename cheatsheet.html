Enumerable: Map

Ruby's #map method is good for iterating over an array. For example:

[1,2,3,4].map{|x| puts x} => 1,2,3,4

Enumerable: Collect

Ruby's #collect method returns values that fit whatever condition is set inside of the block.

[1,2,3,4].collect{ |x| x + 1 } => [2,3,4,5]

Enumerable: any?

Ruby's #any? method if any element returns true for the paramters set in the block, returns true. Else, it returns false.

%w[ant bear cat].any? { |word| word.length >= 3 } => True

Javascript: For Loops

Javascript's For Loops takes conditions and runs until a condition is specified.

var cars = ["Honda","Nissan","Ford"]
for (i = 0; i < cars.length; i++) {
text += cars[i];
}

Javascript: Objects

Javascript's objects with an attribute are similar to hashes in Ruby.

var anything = {
something: 1,
something_else: 2;
}

Javascript: Functions

Javascript's Functions are a method that runs a certain block of code.
function sum(array) {
var total = 0;
for (var i = 0; i < array.length; i++) {
total += array[i];
}
